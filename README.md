# Windows IT Support & Troubleshooting Lab

## Overview

This project demonstrates foundational Windows administration and troubleshooting skills through hands-on use of native Windows tools.

## Skills Demonstrated

- Windows Administration
- Technical Troubleshooting
- Network Diagnostics
- User Account Management
- Hardware and Driver Inspection
- Event Log Analysis
- Performance Monitoring
- Technical Documentation

## Tools Used

- Windows 11
- System Information
- Command Prompt
- Device Manager
- Event Viewer
- Task Manager

## Network Commands Used

- `ipconfig`
- `ipconfig /all`
- `ping`
- `nslookup`
- `tracert`

## Tasks Completed

- Gathered operating system and hardware information
- Reviewed Windows user-account settings
- Performed basic network connectivity and DNS tests
- Inspected device status and driver information
- Reviewed Windows system logs
- Monitored CPU, memory, disk, and startup activity
- Documented each task with explanations and screenshots
